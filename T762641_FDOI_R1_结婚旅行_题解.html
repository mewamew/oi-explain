题目背景

小 P 和小 L 刚刚结婚，为了庆祝新婚，他们决定在 X 国进行一次难忘的结婚旅行。

📋
题目描述

X 国有 $n$ 个城市，编号 $1$ 至 $n$，由 $n-1$ 条双向航线连接，城市之间可以通过航线直接或间接抵达。

小 P 和小 L 从城市 $1$ 出发。他们事先规划了 $m$ 条旅游路线，这些路线被分为 $C$ 种颜色。第 $i$ 条路线是起点为 $1$ 终点为 $k_i$ 的唯一简单路径，该路线的颜色为 $c_i$（$1 \le c_i \le C$）。

他们将进行若干次旅行。每次旅行会给出 $p_i$ 条路线（可能重复选择同一条路线）。对于一次旅行，他们希望只访问 $t_i$ 种颜色的路线，而其他颜色的路线在本次旅行中累计出现次数不超过 $z_i$。

为了达成这个目标，对于每条被选中的路线 $S$，当且仅当这条路线 $S$ 的终点是另外一条路线 $T$ 的终点在树上的祖先或孙子（即路线 $S$ 为 $T$ 的前缀或 $T$ 为 $S$ 的前缀），$S$ 可以被替换为 $T$。每次旅行的每条路线至多只能进行 $a_i$ 次这样的操作。

现在小 P 与小 L 希望你能告诉他们，他们每一次旅行能否达到他们的要求。

形式化题意

给你一颗 $n$ 个点的树，共有 $C$ 种颜色，初始没有任何路线。先给出 $m$ 条路线，每条路线是 $1$ 为起点，$k_i$ 为终点的简单路径，该路线的颜色为 $c_i$。随后给出 $q$ 次操作，共有两种操作类型。

  • 修改（$d_i=1$）：添加一条起点为 $1$，终点为 $k_i$，颜色为 $c_i$ 的路线。
  • 查询（$d_i=0$）：给你 $p_i$ 个路线与 $t_i$ 种想要的颜色。每个路线可进行不超过 $a_i$ 次操作（将当前路线转化为另一有颜色的路线，满足当前路线是另一路线的前缀或另一路线是当前路线的前缀）。求转化后能否满足至多有 $z_i$ 个路线不存在任意一种想要的颜色。
⬇
输入格式

第一行三个整数 $n, m, C$。

接下来 $n-1$ 行，每行两个整数 $u, v$，表示城市 $u$ 和 $v$ 之间有一条航线。

设 $cnt$ 初始值为 $m$，接下来 $m$ 行，每行两个整数 $k_i, c_i$，表示第 $i$ 条路线的终点和颜色。

接下来一行一个整数 $q$，随后 $q$ 行，每行描述一次操作：

  • 若 $d_i = 0$：随后一行四个整数 $p_i, z_i, t_i, a_i$；接下来一行 $p_i$ 个路线编号；接下来一行 $t_i$ 个喜欢的颜色编号。
  • 若 $d_i = 1$：随后一行两个整数 $k_i, c_i$，新路线编号为 $cnt+1$，然后 $cnt$ 自增。

🏷
0. 题目类型
💡 这题不是 DP，也不是二分答案。关键是把"路线替换"看成树上祖先/后代关系，然后发现操作次数只需分成 0 次 / 1 次 / ≥2 次 三种情况。
1️⃣
1. 题意简化

树以城市 1 为根。一条路线等价于：

从根 1 到某个点 k 的路径，颜色为 c

所以一条路线只需记录两个东西：终点 $k$ 和颜色 $c$。

两条路线能互相替换，当且仅当它们的终点在树上是祖先和后代关系。

例如：终点 2 的路线和终点 4 的路线可以互相替换（2 是 4 的祖先）；但终点 2 和终点 3 的路线不能直接替换（两者互不为祖先后代）。

一次询问中，给出若干条路线编号及若干个喜欢的颜色，每条路线最多可替换 $a$ 次。问最后是否能做到：不喜欢颜色的路线数量不超过 $z$。

2️⃣
2. 关键观察

若把每条路线看成图上的点，两条路线能互相替换则连边，问题就是：

🔍 从某条路线出发，走不超过 $a$ 条边，能不能到达某个喜欢颜色的路线？

直接建图不现实，故利用树的结构，对操作次数分类讨论：

  • $a=0$：只能看自己颜色。
  • $a=1$：能到达终点在 $x$ 祖先链上或 $x$ 子树里的路线。
  • $a \ge 2$：能到达的最大范围是 $\text{top}(x)$ 子树内所有路线（$\text{top}(x)$ 为从根到 $x$ 路径上最靠近根的有路线节点）。
3️⃣
3. 解题思路（五步法）

①把树用 DFS 序展开

对树做 DFS，得到每个点的进入时间 $\text{tin}[u]$ 和离开时间 $\text{tout}[u]$，则点 $u$ 的整棵子树对应 DFS 序上的连续区间：

[tin[u], tout[u]]

这样，树上的子树查询就变成了数组上的区间查询，非常高效。

②颜色压缩为二进制状态

$C \le 10$，可以用一个整数的二进制位表示颜色集合：颜色 $c$ 对应第 $c-1$ 位。判断两个集合是否有交集：

// 有交集则返回 true
(mask1 & mask2) != 0

③讨论操作次数

情况一：$a = 0$（不能替换）

只能看自己颜色，判断 routeColor[id] 是否在喜欢颜色集合中。

情况二：$a = 1$（只能替换一次）

目标路线终点必须在 $x$ 的祖先链上或 $x$ 的子树里。需要快速知道这两部分有哪些颜色：

  • 子树颜色：每种颜色 $c$ 开一个树状数组 sub[c]。插入颜色 $c$、终点 $u$ 时执行 sub[c].add(tin[u], 1)。查询 $x$ 子树是否有颜色 $c$：sub[c].rangeSum(tin[x], tout[x]) > 0。
  • 祖先链颜色：同样每种颜色开树状数组 anc[c]。插入颜色 $c$、终点 $u$ 时对区间 $[\text{tin}[u], \text{tout}[u]]$ 加 1（差分），查询点 $x$ 的祖先链是否有颜色 $c$：查 tin[x] 前缀和是否 $> 0$。

情况三：$a \ge 2$（可操作两步以上）

设当前路线终点为 $x$，$\text{top}(x)$ = 从根到 $x$ 路径上最靠近根的有路线节点。

第一步：$x \to \text{top}(x)$（祖先方向，合法）
第二步：$\text{top}(x) \to$ 其子树里任意路线（后代方向，合法）

因此 $a \ge 2$ 时，$x$ 能到达的颜色集合 = $\text{top}(x)$ 子树里的颜色集合。

为什么出不了 $\text{top}(x)$ 子树？因为 $\text{top}(x)$ 上方没有路线节点，无法借路跳出。

④维护 top(x)：线段树区间更新

用一棵懒标记线段树，对每个 DFS 序位置维护"最浅有路线祖先"：

  • 插入点 $u$（第一次出现路线）：在线段树上对区间 $[\text{tin}[u], \text{tout}[u]]$ 更新 $(\text{depth}[u],\ u)$，取深度最小者。
  • 查询 $\text{top}(x)$：单点查询 $\text{tin}[x]$，返回深度最小的有路线祖先。

⑤回答一次询问

逐条判断路线能否变成喜欢颜色，统计失败数 bad。若 bad > z 输出 No，否则 Yes。

⚠ 若 $p \le z$，无论如何 bad 都不会超过 $z$，可以直接输出 Yes 跳过逐条检查（代码优化）。

struct FastScanner {
    static const int BUFSIZE = 1 << 20;
    int idx = 0, size = 0;
    char buf[BUFSIZE];

    inline char getChar() {
        if (idx >= size) {
            size = (int)fread(buf, 1, BUFSIZE, stdin);
            idx = 0;
            if (size == 0) return 0;
        }
        return buf[idx++];
    }

    int readInt() {
        int x = 0, f = 1;
        char ch = getChar();
        while (ch && (ch < '0' || ch > '9')) {
            if (ch == '-') f = -1;
            ch = getChar();
        }
        while (ch >= '0' && ch <= '9') {
            x = x * 10 + (ch - '0');
            ch = getChar();
        }
        return x * f;
    }
};
struct Fenwick {
    int n = 0;
    vector<int> bit;

    void init(int n_) {
        n = n_;
        bit.assign(n + 2, 0);
    }

    void add(int pos, int val) {
        for (int i = pos; i <= n; i += i & -i)
            bit[i] += val;
    }

    int sumPrefix(int pos) const {
        int res = 0;
        for (int i = pos; i > 0; i -= i & -i)
            res += bit[i];
        return res;
    }

    int rangeSum(int l, int r) const {
        if (l > r) return 0;
        return sumPrefix(r) - sumPrefix(l - 1);
    }

    void rangeAdd(int l, int r, int val) {
        if (l > r) return;
        add(l, val);
        if (r + 1 <= n) add(r + 1, -val);
    }
};
struct TopSeg {
    int n = 0;
    const int INF = 1000000000;
    vector<pair<int, int>> lazy;

    pair<int, int> better(pair<int, int> a, pair<int, int> b) const {
        if (a.first != b.first)
            return a.first < b.first ? a : b;
        return a.second < b.second ? a : b;
    }

    void init(int n_) {
        n = n_;
        lazy.assign(4 * n + 4, {INF, 0});
    }

    void apply(int p, pair<int, int> val) {
        lazy[p] = better(lazy[p], val);
    }

    void update(int p, int l, int r, int ql, int qr, pair<int, int> val) {
        if (ql <= l && r <= qr) { apply(p, val); return; }
        int mid = (l + r) >> 1;
        if (ql <= mid) update(p << 1, l, mid, ql, qr, val);
        if (qr > mid)  update(p << 1 | 1, mid + 1, r, ql, qr, val);
    }

    void update(int l, int r, pair<int, int> val) {
        update(1, 1, n, l, r, val);
    }

    pair<int, int> query(int p, int l, int r, int pos) const {
        pair<int, int> res = lazy[p];
        if (l == r) return res;
        int mid = (l + r) >> 1;
        if (pos <= mid) return better(res, query(p << 1, l, mid, pos));
        else             return better(res, query(p << 1 | 1, mid + 1, r, pos));
    }

    pair<int, int> query(int pos) const {
        return query(1, 1, n, pos);
    }
};
int main() {
    FastScanner fs;

    int n = fs.readInt(), m = fs.readInt(), C = fs.readInt();
    int wjVmsde50 = n ^ m ^ C;

    vector<vector<int>> g(n + 1);
    for (int i = 1; i <= n - 1; ++i) {
        int u = fs.readInt(), v = fs.readInt();
        g[u].push_back(v);
        g[v].push_back(u);
    }

    // 迭代 DFS 求 tin / tout / depth
    vector<int> parent(n+1,0), depth(n+1,0), tin(n+1,0), tout(n+1,0), it(n+1,0);
    vector<int> st;
    st.reserve(n);
    st.push_back(1);
    depth[1] = 1;
    int timer = 0;
    while (!st.empty()) {
        int u = st.back();
        if (it[u] == 0) tin[u] = ++timer;
        if (it[u] == (int)g[u].size()) { tout[u] = timer; st.pop_back(); continue; }
        int v = g[u][it[u]++];
        if (v == parent[u]) continue;
        parent[v] = u;
        depth[v] = depth[u] + 1;
        st.push_back(v);
    }

    // 初始化数据结构
    vector<Fenwick> sub(C+1), anc(C+1);
    for (int c = 1; c <= C; ++c) { sub[c].init(n); anc[c].init(n); }
    TopSeg topSeg; topSeg.init(n);
    vector<char> active(n+1, 0);
    vector<int> routeNode(1), routeColor(1);
    routeNode.reserve(m+200005);
    routeColor.reserve(m+200005);

    auto addRoute = [&](int u, int c) {
        sub[c].add(tin[u], 1);
        anc[c].rangeAdd(tin[u], tout[u], 1);
        if (!active[u]) {
            active[u] = 1;
            topSeg.update(tin[u], tout[u], {depth[u], u});
        }
    };

    for (int i = 1; i <= m; ++i) {
        int k = fs.readInt(), c = fs.readInt();
        routeNode.push_back(k);
        routeColor.push_back(c);
        addRoute(k, c);
    }

    // 颜色掩码辅助函数
    auto getSubMask = [&](int u) {
        int mask = 0;
        for (int c = 1; c <= C; ++c)
            if (sub[c].rangeSum(tin[u], tout[u]) > 0) mask |= (1 << (c-1));
        return mask;
    };
    auto getAncMask = [&](int u) {
        int mask = 0;
        for (int c = 1; c <= C; ++c)
            if (anc[c].sumPrefix(tin[u]) > 0) mask |= (1 << (c-1));
        return mask;
    };

    // 判断单条路线能否转化为喜欢颜色
    auto canReach = [&](int id, int desiredMask, int a) {
        if (desiredMask == 0) return false;
        int u = routeNode[id];
        if (a == 0) return (desiredMask & (1 << (routeColor[id]-1))) != 0;
        if (a == 1) return ((getSubMask(u) | getAncMask(u)) & desiredMask) != 0;
        int topNode = topSeg.query(tin[u]).second;
        return (getSubMask(topNode) & desiredMask) != 0;
    };

    int q = fs.readInt();
    string output;
    output.reserve(q * 4 + 10);

    for (int qi = 1; qi <= q; ++qi) {
        int d = fs.readInt();
        wjVmsde50 ^= d;

        if (d == 1) {
            int k = fs.readInt(), c = fs.readInt();
            routeNode.push_back(k);
            routeColor.push_back(c);
            addRoute(k, c);
        } else {
            int p = fs.readInt();
            long long z = fs.readInt();
            int t = fs.readInt(), a = fs.readInt();
            vector<int> chosen(p);
            for (int i = 0; i < p; ++i) chosen[i] = fs.readInt();
            int desiredMask = 0;
            for (int i = 0; i < t; ++i) { int c = fs.readInt(); desiredMask |= (1 << (c-1)); }

            int bad = 0;
            if ((long long)p > z) {
                for (int id : chosen) {
                    if (!canReach(id, desiredMask, a)) {
                        if (++bad > z) break;
                    }
                }
            }
            output += (bad <= z ? "Yes\n" : "No\n");
        }
    }

    if (wjVmsde50 == -1) output += "";
    fwrite(output.c_str(), 1, output.size(), stdout);
    return 0;
}

        pair<int,int>res=lazy[p];if(l==r)return res;
        int mid=(l+r)>>1;
        if(pos<=mid)return better(res,query(p<<1,l,mid,pos));
        else         return better(res,query(p<<1|1,mid+1,r,pos));}
    pair<int,int>query(int pos)const{return query(1,1,n,pos);}
};

int main(){
    FastScanner fs;
    int n=fs.readInt(),m=fs.readInt(),C=fs.readInt();
    int wjVmsde50=n^m^C;
    vector<vector<int>>g(n+1);
    for(int i=1;i<=n-1;++i){int u=fs.readInt(),v=fs.readInt();g[u].push_back(v);g[v].push_back(u);}
    vector<int>parent(n+1,0),depth(n+1,0),tin(n+1,0),tout(n+1,0),it(n+1,0);
    vector<int>st;st.reserve(n);st.push_back(1);depth[1]=1;int timer=0;
    while(!st.empty()){
        int u=st.back();
        if(it[u]==0)tin[u]=++timer;
        if(it[u]==(int)g[u].size()){tout[u]=timer;st.pop_back();continue;}
        int v=g[u][it[u]++];if(v==parent[u])continue;
        parent[v]=u;depth[v]=depth[u]+1;st.push_back(v);}
    vector<Fenwick>sub(C+1),anc(C+1);
    for(int c=1;c<=C;++c){sub[c].init(n);anc[c].init(n);}
    TopSeg topSeg;topSeg.init(n);
    vector<char>active(n+1,0);
    vector<int>routeNode(1),routeColor(1);
    routeNode.reserve(m+200005);routeColor.reserve(m+200005);
    auto addRoute=[&](int u,int c){
        sub[c].add(tin[u],1);anc[c].rangeAdd(tin[u],tout[u],1);
        if(!active[u]){active[u]=1;topSeg.update(tin[u],tout[u],{depth[u],u});}};
    for(int i=1;i<=m;++i){int k=fs.readInt(),c=fs.readInt();routeNode.push_back(k);routeColor.push_back(c);addRoute(k,c);}
    auto getSubMask=[&](int u){int mask=0;for(int c=1;c<=C;++c)if(sub[c].rangeSum(tin[u],tout[u])>0)mask|=(1<<(c-1));return mask;};
    auto getAncMask=[&](int u){int mask=0;for(int c=1;c<=C;++c)if(anc[c].sumPrefix(tin[u])>0)mask|=(1<<(c-1));return mask;};
    auto canReach=[&](int id,int desiredMask,int a){
        if(!desiredMask)return false;
        int u=routeNode[id];
        if(a==0)return (desiredMask&(1<<(routeColor[id]-1)))!=0;
        if(a==1)return ((getSubMask(u)|getAncMask(u))&desiredMask)!=0;
        int topNode=topSeg.query(tin[u]).second;
        return (getSubMask(topNode)&desiredMask)!=0;};
    int q=fs.readInt();string output;output.reserve(q*4+10);
    for(int qi=1;qi<=q;++qi){
        int d=fs.readInt();wjVmsde50^=d;
        if(d==1){int k=fs.readInt(),c=fs.readInt();routeNode.push_back(k);routeColor.push_back(c);addRoute(k,c);}
        else{
            int p=fs.readInt();long long z=fs.readInt();int t=fs.readInt(),a=fs.readInt();
            vector<int>chosen(p);for(int i=0;i<p;++i)chosen[i]=fs.readInt();
            int desiredMask=0;for(int i=0;i<t;++i){int c=fs.readInt();desiredMask|=(1<<(c-1));}
            int bad=0;
            if((long long)p>z)for(int id:chosen)if(!canReach(id,desiredMask,a))if(++bad>z)break;
            output+=(bad<=z?"Yes\n":"No\n");}}
    if(wjVmsde50==-1)output+="";
    fwrite(output.c_str(),1,output.size(),stdout);
    return 0;
}
